# Roadmap

## Current Version (v1.0) — MVP ✅

- [x] Lead management (create, edit, filter, grade)
- [x] Customer management (360° view)
- [x] Contact management
- [x] Business line management
- [x] Project/Opportunity tracking
- [x] Follow-up records and templates
- [x] Task management
- [x] Quote management with line items
- [x] Order management with status flow
- [x] Product catalog
- [x] Document management
- [x] CSV import/export
- [x] Activity logging
- [x] Global search
- [x] AI analysis (OpenAI-compatible)

- [x] Dark sidebar UI with modern design
