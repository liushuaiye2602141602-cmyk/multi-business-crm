# 系统要求

## 软件要求

| 软件 | 最低版本 | 推荐版本 |
|------|---------|---------|
| Node.js | 18.0 | 20.x LTS |
| npm | 9.0 | 10.x |
| PostgreSQL | 14.0 | 16.x |
| Docker | 20.10 | 最新稳定版 |
| Docker Compose | 2.0 | 最新稳定版 |

## 硬件要求

| 环境 | CPU | 内存 | 磁盘 |
|------|-----|------|------|
| 开发环境 | 2 核 | 4 GB | 5 GB |
| 生产环境 | 4 核 | 8 GB | 20 GB |

## 操作系统

- Windows 10/11（推荐 PowerShell）
- Ubuntu 20.04+ / Debian 11+
- macOS 12+

## 浏览器

- Chrome 90+
- Firefox 90+
- Edge 90+
- Safari 15+

## 网络

- 开发环境：仅需本地网络
- 生产环境：需公网访问（用于 Webhook 和 IM 机器人）
