# Release v0.1.0 — Open Source Initial Release

**Release Date:** 2026-06-18
**Status:** Stable

## 🎉 Welcome to Open CRM System

Open CRM System is a modern, open-source CRM template built with Next.js, TypeScript, Prisma, and PostgreSQL. Designed for flexibility — customize it for any industry or business model.

## 📦 What's Included

### Lead & Customer Management
- Lead pipeline with status tracking
- Customer profiles with 360° view
- Contact management
- Business line segmentation
- Customer public pool
- Dormant customer detection

### Sales Pipeline
- Project/opportunity tracking
- Quote management with line items
- Order management with full status flow
- Task management

### Finance
- Invoice management
- Payment tracking
- Financial overview

### AI Integration
- OpenAI-compatible API
- 16 natural language intents
- Vision AI for screenshots
- Auto-analyze leads

### IM Integration
- Feishu bot (long connection)
- Natural language commands
- Screenshot recognition

### Email
- SMTP/IMAP support
- Email statistics

### Reports
- Sales funnel, trends, distribution charts
- Business line comparison

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 16, React 19, Tailwind CSS 4 |
| Backend | Next.js API Routes, Prisma ORM |
| Database | PostgreSQL (Docker) |
| Charts | Recharts |
| AI | OpenAI-compatible API |
| Email | Nodemailer + ImapFlow |

## 🚀 Quick Start

```bash
git clone https://github.com/liushuaiye2602141602-cmyk/multi-business-crm.git
cd multi-business-crm
npm install
cp .env.example .env
# Edit .env with your database password
docker run -d --name open-crm-postgres -e POSTGRES_PASSWORD=your_password -e POSTGRES_DB=open_crm -p 5433:5432 postgres:16
npx prisma db push
npx prisma generate
npm run dev
```

Visit http://localhost:3003

## ⚠️ Known Limitations

- No user authentication (system is accessible without login)
- Single-tenant only (no multi-user isolation)
- Feishu integration requires a Feishu developer account
- AI features require an OpenAI-compatible API key
- Email features require SMTP/IMAP server access
