## 当前文件存储状态

**项目当前没有任何文件上传/存储功能。**

- `Document` 模型有 `fileUrl` 和 `fileName` 字段，但没有对应的上传实现
- `Email` 模型没有附件字段
- 没有任何文件写入代码
- 没有对象存储集成
