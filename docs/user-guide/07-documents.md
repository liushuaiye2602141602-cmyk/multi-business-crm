# 文档管理

> **当前版本部分支持：** 本模块当前仅支持文档元数据的记录与管理（标题、类型、URL 引用等），文件上传与存储功能尚未实现。实际的文件存储和上传功能将在后续版本中提供。

## 概述

文档管理模块用于记录和管理外贸业务过程中产生的各类文档信息。您可以为报价、合同、设计稿、产品规格等建立文档记录，并将其关联到对应的客户、项目、报价或订单。

## 页面入口

| 页面 | 路由 | 说明 |
|------|------|------|
| 文档列表 | `/documents` | 查看所有文档记录 |
| 新建文档 | `/documents/new` | 创建新的文档记录 |
| 文档详情 | `/documents/[id]` | 查看文档详细信息 |
| 编辑文档 | `/documents/[id]/edit` | 修改文档信息 |

## 文档字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| title | String | 文档标题 |
| documentType | Enum | 文档类型（见下方枚举值） |
| fileUrl | String | 文件 URL 引用地址（当前版本手动填写外部链接） |
| fileName | String | 文件名称 |
| notes | String | 备注说明 |
| relatedType | Enum | 关联实体类型（见下方枚举值） |
| relatedId | String | 关联实体的 ID |

## 文档类型（DocumentType）

| 值 | 中文说明 |
|------|------|
| QUOTE | 报价单 |
| CONTRACT | 合同 |
| ARTWORK | 设计稿 |
| SAMPLE_PHOTO | 样品照片 |
| PRODUCT_SPEC | 产品规格 |
| PACKING_REQUIREMENT | 包装要求 |
| OTHER | 其他 |

## 关联实体类型（DocumentRelatedType）

| 值 | 中文说明 |
|------|------|
| CUSTOMER | 客户 |
| PROJECT | 项目 |
| QUOTE | 报价 |
| ORDER | 订单 |

## 使用说明

### 创建文档记录

1. 进入文档列表页面（`/documents`），点击「新建文档」按钮。
2. 填写文档标题、选择文档类型。
3. 在 fileUrl 字段中填入文件的外部存储链接（如网盘链接、在线文档地址等）。
4. 填写文件名称，便于识别。
5. 在「关联实体」部分，选择关联类型（客户/项目/报价/订单），并填入对应的实体 ID。
6. 添加备注信息（可选）。
7. 保存文档记录。

### 关联实体说明

文档通过 `relatedType` + `relatedId` 的组合模式关联到业务实体：

- **关联到客户（CUSTOMER）：** 将文档绑定到特定客户，如客户的产品规格书。
- **关联到项目（PROJECT）：** 将文档绑定到项目，如项目合同或设计稿。
- **关联到报价（QUOTE）：** 将文档绑定到报价单，如报价单附件。
- **关联到订单（ORDER）：** 将文档绑定到订单，如装箱要求或合同文件。

### 查看和编辑文档

- 在文档列表中点击文档标题可进入详情页面查看完整信息。
- 在详情页面点击「编辑」可修改文档的元数据信息。

### 搜索和筛选

文档列表支持按标题、文档类型、关联实体类型等条件进行搜索和筛选，帮助您快速定位所需的文档记录。

## 注意事项

- 当前版本不支持直接上传文件，fileUrl 需填写外部存储地址。
- 删除文档记录后，关联关系将同时解除。
- 建议使用规范的文件命名和统一的外部存储方案，以便于文档管理。
