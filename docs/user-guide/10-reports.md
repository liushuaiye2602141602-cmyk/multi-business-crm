# 报表与分析

## 概述

报表与分析模块提供多维度的业务数据统计和可视化分析，帮助管理者全面了解外贸业务运营状况，做出数据驱动的决策。系统使用 Recharts 图表库进行数据可视化，数据来源于 Prisma 聚合查询。

## 页面入口

| 页面 | 路由 | 说明 |
|------|------|------|
| 报表中心 | `/reports` | 报表与分析主页面 |
| 销售目标 | `/goals` | 销售目标跟踪与管理 |

## 报表类型

系统提供 6 种报表类型：

### 1. 漏斗报告（Funnel Report）

**路径：** 报表中心 → 漏斗报告

展示线索转化漏斗，分析各阶段的转化率和流失情况。

**转化阶段：**

```
NEW（新建）→ CONTACTED（已联系）→ QUALIFIED（已确认）→ WON（成交）
```

**分析内容：**

- 各阶段线索数量和占比。
- 阶段间的转化率。
- 各阶段的流失率分析。
- 转化周期统计。

### 2. 趋势报告（Trends Report）

**路径：** 报表中心 → 趋势报告

展示收入和线索的时间趋势变化，支持按月度或季度维度查看。

**分析内容：**

- 月度/季度收入趋势。
- 月度/季度新增线索趋势。
- 成交金额变化趋势。
- 线索增长趋势。

### 3. 分布报告（Distribution Report）

**路径：** 报表中心 → 分布报告

展示线索和客户的地理分布与行业分布情况。

**分析内容：**

- 线索/客户按国家或地区的分布。
- 线索/客户按行业的分布。
- 各地区/行业的成交情况对比。
- 热门市场和潜力市场分析。

### 4. 来源报告（Source Report）

**路径：** 报表中心 → 来源报告

分析各线索来源的效果，帮助优化获客渠道投入。

**来源类型：**

| 值 | 说明 |
|------|------|
| WEBSITE | 官网 |
| FACEBOOK | Facebook 广告 |
| TIKTOK | TikTok 广告 |
| ALIBABA | 阿里巴巴国际站 |
| MADE_IN_CHINA | Made-in-China |
| TRADE_SHOW | 展会 |
| REFERRAL | 客户转介 |
| COLD_CALL | 电话开发 |
| OTHER | 其他 |

**分析内容：**

- 各来源的线索数量和占比。
- 各来源的转化率和成交率。
- 各来源的获客成本分析。
- 各来源的线索质量评估。

### 5. 业务线报告（Business Line Report）

**路径：** 报表中心 → 业务线报告

对比各业务线的业绩表现，帮助进行资源分配和业务决策。

**分析内容：**

- 各业务线的收入对比。
- 各业务线的线索数量和成交率。
- 各业务线的客户数量增长。
- 业务线间的业绩排名。

### 6. 跟进报告（Follow-up Report）

**路径：** 报表中心 → 跟进报告

统计和分析跟进活动的数量及效果。

**分析内容：**

- 各销售人员的跟进频率和数量。
- 跟进方式分布（邮件、电话、WhatsApp 等）。
- 跟进活动与成交的关联分析。
- 跟进响应时间和客户反馈统计。

## 销售目标（SalesGoal）

**路径：** `/goals`

### 目标字段

| 字段 | 类型 | 说明 |
|------|------|------|
| year | Number | 目标年份 |
| month | Number | 目标月份 |
| metricType | Enum | 指标类型（收入、线索数、成交数等） |
| targetValue | Number | 目标值 |
| currentValue | Number | 当前完成值 |
| currency | String | 货币单位 |

### 目标管理说明

- 同一年份、月份、指标类型下只能创建一个目标（唯一约束）。
- 系统自动计算目标完成率（当前值 / 目标值）。
- 支持按月度和年度维度查看目标完成情况。
- 目标数据与报表模块联动，提供实时更新。

## 数据可视化

所有报表均使用 Recharts 图表库进行可视化展示，支持：

- **柱状图（Bar Chart）** — 用于对比分析（如各来源线索数、各业务线收入）。
- **折线图（Line Chart）** — 用于趋势分析（如月度收入趋势）。
- **漏斗图（Funnel Chart）** — 用于转化分析（如线索转化漏斗）。
- **饼图（Pie Chart）** — 用于分布分析（如地区分布、行业分布）。

## 使用说明

### 查看报表

1. 进入报表中心页面（`/reports`）。
2. 从顶部导航选择所需的报表类型。
3. 设置筛选条件（时间范围、业务线等）。
4. 查看图表和详细数据。
5. 支持导出报表数据。

### 设置销售目标

1. 进入销售目标页面（`/goals`）。
2. 点击「新建目标」。
3. 选择目标年份和月份。
4. 选择指标类型，填写目标值。
5. 选择货币单位。
6. 保存目标。

## 注意事项

- 报表数据基于数据库中的实际记录进行聚合计算，数据更新可能有短暂延迟。
- 部分报表需要足够的数据量才能呈现有意义的分析结果。
- 建议定期查看趋势报告，及时掌握业务变化。
- 销售目标建议在每月初设定，以便进行整月跟踪。
